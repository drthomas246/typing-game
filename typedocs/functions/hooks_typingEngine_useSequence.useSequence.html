    • useSequence(
          QA: QAPair[],
          opts: EngineOptions,

          getPair: (index: number) => QAPair;
          initOrder: () => void;

      タイピングゲームの問題の出題順序を管理するカスタムフック。 問題セット (QAPair[]) とオプション (EngineOptions) に基づいて、 問題の順序を初期化（ランダムまたはシーケンシャル）し、指定されたインデックスの問題を取得する機能を提供します。

      Parameters

      • QA: QAPair[]

        問題データの配列。

      • opts: EngineOptions

        エンジンのオプション。特にseedとrandomOrderが使用されます。

        タイピングエンジンの動作オプション。

        • OptionalbattleMode?: boolean

          バトルモードを有効にするか。

        • Optionalbgm?: boolean

          BGMを有効にするか。

        • OptionalbgmSrc?: string

          BGMファイルのソースパス。

        • OptionalbgmVolume?: number

          BGMの音量。

        • OptionaldamagePerHit?: number

          正解1文字あたりのダメージ量（敵へのダメージ？）。

        • OptionaldamagePerMiss?: number

          ミス1回あたりのダメージ量。

        • OptionaldamagePerSentence?: number

          1文完了あたりのダメージ量。

        • OptionalenemyMaxHp?: number

          敵の最大HP。

        • OptionallearningMode?: boolean

          学習モードを有効にするか。

        • OptionallearnThenRecall?: boolean

          学習フェーズ後に復習フェーズ（Learn Then Recall）を行うか。

        • OptionalplayerMaxHp?: number

          プレイヤーの最大HP。

        • OptionalrandomOrder?: boolean

          問題をランダムな順序にするか。

        • Optionalseed?: number

          乱数シード。

        • Optionalsfx?: boolean

          効果音（SFX）を有効にするか。

        • OptionalsfxDefeatSrc?: string

          敗北効果音のソースパス。

        • OptionalsfxEscapeSrc?: string

          逃走効果音のソースパス。

        • OptionalsfxFallDownSrc?: string

          転倒効果音のソースパス。

        • OptionalsfxKeyOnSrc?: string

          キー入力効果音のソースパス。

        • OptionalsfxLevelUpSrc?: string

          レベルアップ効果音のソースパス。

        • OptionalsfxPunchSrc?: string

          パンチ効果音のソースパス。

        • OptionalsfxSlashSrc?: string

          斬撃効果音のソースパス。

        • OptionalsfxVolume?: number

          効果音の音量。

        • Optionalsound?: boolean

          全体的なサウンドを有効にするか。

        • OptionaltickMs?: number

          エンジンティックの間隔（ミリ秒）。

      Returns { getPair: (index: number) => QAPair; initOrder: () => void; order: number[] }

      問題の順序、初期化関数、問題取得関数を含むオブジェクト。
